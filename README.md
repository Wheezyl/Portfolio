# Portfolio Alternance – BUT Réseaux & Télécommunications

Ce dépôt contient le code source de mon **portfolio d’alternance** réalisé dans le cadre de mon **BUT Réseaux & Télécommunications**.  
Ce site a pour objectif de présenter mon **entreprise d’accueil**, les **missions réalisées**, ainsi que mon **bilan de compétences** en lien avec le référentiel de formation.

---

##  Objectifs du site

- Présenter mon parcours en alternance de manière claire et structurée
- Mettre en valeur les missions techniques réalisées en entreprise
- Faire le lien entre les compétences du BUT R&T et la pratique professionnelle
- Proposer un support lisible pour l’évaluation du portfolio

---

##  Structure du site

Le site est composé de **7 pages HTML** et **1 feuille de style CSS** :

- `index.html` : page d’accueil et présentation générale
- `entreprise.html` : contexte de l’entreprise et rôle occupé
- `missions.html` : vue d’ensemble des missions
- `mission-switches.html` : déploiement et configuration de switches
- `mission-wifi.html` : installation et gestion des bornes WiFi
- `mission-support.html` : support réseau et incidents terrain
- `bilans.html` : bilan de compétences en lien avec le BUT R&T
- `styles.css` : feuille de style commune à l’ensemble du site

---

##  Technologies utilisées

- **HTML5** (structure sémantique)
- **CSS3** (mise en page, responsive design)
- **Google Fonts** (Poppins)
- Design responsive adapté aux écrans mobiles et desktop

---

##  Organisation du code

- Une structure simple et lisible
- Un style CSS unique partagé par toutes les pages
- Une navigation cohérente entre les pages
- Contenu organisé par sections (missions, compétences, bilans)
